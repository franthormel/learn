# [auto_route](https://pub.dev/packages/auto_route)

Flutter example for using the **`auto_route`** package
This sample is divider into two (2) parts:

### **I. Basics**

    Navigation
    Passing Arguments
    Page Transitions

### **II. Advanced**

    Nested Routes
    Wrapping Routes
    Declarative Routing
    Bottom Navigation Bar
    Route Guards
    Authentication
    Firebase Analytics
