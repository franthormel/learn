# Variables

Used to store values.
Fields are members of a class.

### Instance
```
public class Bike {
    int cadence = 0;
    int speed = 0;
    int gear = 0;
}
```

### Static
```
public class Bike {
    static int seat = 1;
    static final int NUM_GEARS = 6;
}
```

### Local
```
int count = 0;
```

### Parameters
```
public static void main(String[]) args
```
